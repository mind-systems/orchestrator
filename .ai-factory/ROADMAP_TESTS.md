## agents.py

- [x] **`_has_signal` unit tests** — `tests/test_agents.py`. Import: `from orchestrator.agents import _has_signal` (`agents.py:42`). Implementation: `any(line.strip() == signal for line in text.splitlines()[-5:])` (`agents.py:44`) — window is exactly the last 5 lines, comparison is `.strip() == signal` (exact match). Window boundary (10-line text): `splitlines()[-5:]` = indices 5–9 = lines 6–10 (1-indexed); line 5 (index 4) is the first line **excluded**. Cases: (1) exact signal on last line → True; (2) exact signal on line 3 of 5 (non-last line, still inside window for 5-line text) → True; (3) signal on line 5 of a 10-line text (just outside the last-5 window, index 4) → False; (4) signal on line 6 of a 10-line text (first line inside the window, index 5) → True; (5) signal as substring of longer word (`"no REVIEW_PASS here"`) → False (strip+equality rejects this); (6) empty text → False; (7) line has trailing whitespace (`"  REVIEW_PASS  "`) → True (`.strip()` catches it); (8) `PLAN_REVIEW_PASS` signal → True. [3m 9s]
